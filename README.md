# Accounting Dashboard Starter

A professional starter dashboard built with Next.js (App Router), TypeScript, and Tailwind CSS.

## Tech Stack

- Next.js 16 (App Router)
- TypeScript
- Tailwind CSS 4
- ESLint

## Project Structure

```text
src/
  app/
    globals.css
    layout.tsx
    page.tsx
  components/
    layout/
      page-header.tsx
    ui/
      kpi-card.tsx
      section-card.tsx
  features/
    dashboard/
      data/
        summary.ts
  lib/
    utils/
      formatters.ts
  types/
    finance.ts
```

## Run Locally

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.
